##  Simon Game

A simple memory-based Simon Game built using **HTML, CSS, and JavaScript**. The game tests your memory by generating a sequence of colored button flashes that you must repeat in the correct order.

----------------

###  How the Game Works

- The game starts when the user presses any key.
- A random color button will flash ("blink").
- The user must click the same button to proceed.
- After each correct move, the game adds another random button to the sequence.
- The player must repeat the **entire sequence in correct order** each round:
  - First click → correct
  - First + second → correct
  - First + second + third → correct
  - and so on...

- If the user clicks the wrong button, the game ends.
- it will show your score and also attemped


----------------------------

### Game Rules

1. Press any key to start the game.
2. Watch the sequence carefully.
3. Repeat the sequence by clicking the buttons in the correct order.
4. Each level increases difficulty by adding one more step.
5. Wrong click = Game Over.
